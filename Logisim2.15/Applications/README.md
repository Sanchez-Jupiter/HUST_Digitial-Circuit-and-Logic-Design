# Logisim 应用程序

本文件夹包含 Logisim 电路仿真软件的多个版本和相关运行环境。

## 📱 可执行程序

### 推荐使用
- **logisim-ita-cn-2.15.exe** ⭐
  - 版本：Logisim 2.15.0
  - 语言：中文
  - 平台：Windows
  - 直接双击运行，无需 Java 环境
  - **这是做交通灯实验的推荐版本**

### 其他版本

| 文件 | 版本 | 说明 |
|-----|------|------|
| logisim-win-2.7.1.exe | 2.7.1 | 较旧的 Windows 版本，可作备份 |
| logisim-2.15.0-CN.jar | 2.15.0 | JAR 格式，需要 Java 环境，跨平台兼容 |
| logisim-2.7.1-CN.jar | 2.7.1 | JAR 格式，较旧版本 |
| logisimmac-hust.jar | 2.15.0 | Mac 版本（HUST 定制） |
| cs316.jar | - | 支持库 |

## 🚀 快速开始

### 方法一：直接运行（推荐）
```powershell
.\logisim-ita-cn-2.15.exe
```

### 方法二：使用 JAR（需要 Java）
```powershell
java -jar logisim-2.15.0-CN.jar
```
