# 芯片库 (Component Libraries)

本文件夹包含数字逻辑实验所需的标准 TTL 集成电路库。

## 📚 库文件说明

### 1. 7400-lib/
- **内容**：7400 系列标准 TTL 芯片库
- **用途**：提供基础逻辑门电路组件
- **优势**：完整的 7400 系列支持

### 2. 7400-series-rev1/
- **内容**：7400 系列修订版本 (Rev1)
- **用途**：改进版本的 7400 系列芯片
- **适用**：对库功能有特殊需求时使用

### 3. TTL-74x-Series/
- **内容**：74 系列经典芯片库（原 logisim_74v1）
- **用途**：提供 50+ 个常用的 TTL 标准集成电路
- **路径**：`./TTL-74x-Series/74x/`

## 🔌 常用芯片分类

### 触发器/锁存器
- 74109、74109、7474、7476
- 用途：存储电路、计数器状态保存

### 译码器/编码器
- 74138、74148、74147
- 用途：信号转换、地址译码

### 多路选择器/分配器
- 74151、74157
- 用途：数据选择、多路控制

### 计数器
- 74160、74163、74163v2、74193
- 用途：时间计数、BCD 计数

### 锄存器
- 74273、74374
- 用途：数据暂存、输出驱动

### 算术逻辑单元
- 74181、74182（含子文件夹中的多个版本）
- 用途：算术运算、逻辑运算

### 其他
- 7442、7451、74670、74385
- 用途：驱动电路、编码转换

## 📂 文件夹结构

## 🎯 使用指南

### 导入芯片库到项目
1. 在 Logisim 中打开你的电路项目
2. 点击菜单 `仿真 → 添加组件库`
3. 选择对应的 `.circ` 文件
4. 芯片将出现在左侧工具栏中可供使用

### 推荐使用顺序
1. 优先使用 `TTL-74x-Series` 获得完整的芯片支持
2. 特殊需求时参考其他库版本

## 📖 试验相关

这些库文件主要用于支持以下电路设计：

- **7段数码管驱动电路** → 需要 7段解码器
- **比较器与选择器** → 需要 74151、74157 等
- **BCD计数器** → 需要 74160/74163 等
- **交通灯控制系统** → 综合以上所有组件

## ⚙️ 芯片库文件格式
- 格式：`.circ`（Logisim 电路文件格式）
- 兼容版本：Logisim 2.7.1+
- 编辑：使用 Logisim 直接打开和编辑
